Question-1: What are the different ways to select an element in the DOM?

Answer-1: The different ways to select an element in the DOM are: A. document.getElementById("id"); B. document.getElementsByTagName("tag"); C. document.getElementsByClassName("class");
D. document.querySelector("selector"); E. document.querySelectorAll("selector").

Question-2: What is the difference between innerHTML, innerText, and textContent?

Answer-2: innerHTML sets or gets "HTML content" inside a targeted element, innerText gets or sets "only visible text" without hidden elements inside an element, and textContent sets or gets "all text content" with hidden text too. .

Question-3: What is event delegation in the DOM?

Answer-3: Event delegation is a strategy for a parent element to listen to events occurred on its child elements regardless they are added dynamically or not.

Question-4: What is event bubbling in the DOM?

Answer-4: Event bubbling in the DOM refers to the process of starting an event at the targeted element and propagating upward to its parent elements like the way bubbles inside water travel all the way upward to the surface.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer-5: I can create, add, and remove elements using JavaScript in various ways like: i. To create an element, I can use document.createElement(); ii. To add an element, I can use: a. appendChild() (to add an element at the end); b. prepend() (to add an element at the beginning); c. insertBefore() (to insert an element before another element); iii. To remove an element, I can use: A. remove() (to remove the element directly); B. removeChild() (to remove a child from a parent).
